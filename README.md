# Collage maker

## Built with

- [Svelte](https://svelte.dev/)
- [Konva.js](https://konvajs.org/)

## Installation

### Prerequisites

- Git
- Node.js
- npm

### 1. clone the repository

```sh
git clone https://github.com/lionpuro/collage-maker.git
```

### 2. install dependencies

```sh
npm install
```

## Build

```sh
npm run build
```
